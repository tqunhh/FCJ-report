### Kết quả đạt được tuần 5:

* Hiểu và nắm được các dịch vụ quản lý danh tính & bảo mật trên AWS, bao gồm:  
  * Amazon Cognito
  * AWS Organizations
  * AWS Identity Center 
  * DatabaAmazon Key Management Service (KMS)se
  * AWS Security Hub

* Thực hành AWS Security Hub và nắm được quy trình sử dụng cơ bản: 
  * Kích hoạt Security Hub cho tài khoản AWS
  * Xem dashboard và đánh giá kết quả bảo mật, mức độ tuân thủ 
  
* Thực hành tự động hoá quản lý EC2 bằng Lambda

* Nắm được cách quản lý tài nguyên bằng Tag & Resource Group
  * Sử dụng tag thông qua AWS Management Console
  * Sử dụng tag bằng AWS CLI / dòng lệnh

* Thực hành quản lý quyền truy cập EC2 dựa trên Resource Tag với IAM
  * Tạo IAM User phục vụ phân quyền
  * Tạo IAM Policy với điều kiện kiểm tra tag
  * Tạo IAM Role và thực hiện switch role
  * ...

* Tiến hành họp nhóm, sau đó chọn chủ đề dự án và các chức năng chính của sản phẩm
