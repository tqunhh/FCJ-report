### Week 5 Achievements:

* Understand and grasp identity & security management services on AWS, including:  
  * Amazon Cognito
  * AWS Organizations
  * AWS Identity Center 
  * Amazon Key Management Service (KMS)
  * AWS Security Hub

* Practice AWS Security Hub and grasp basic usage process: 
  * Enable Security Hub for AWS account
  * View dashboard and evaluate security results, compliance level 
  
* Practice automating EC2 management with Lambda

* Grasp how to manage resources using Tag & Resource Group
  * Use tags through AWS Management Console
  * Use tags via AWS CLI / command line

* Practice managing EC2 access based on Resource Tag with IAM
  * Create IAM User for permission assignment
  * Create IAM Policy with tag checking conditions
  * Create IAM Role and perform switch role
  * ...

* Conduct team meeting, then select project topic and main product features
