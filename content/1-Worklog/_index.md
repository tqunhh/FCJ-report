**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)

**Week 2:** [Route 53 and AWS Backup](1.2-week2/)

**Week 3:** [Storage services (S3, Storage Gateway)](1.3-week3/)

**Week 4:** [Storage Gateway and VM migration](1.4-week4/)

**Week 5:** [Security services (IAM, KMS, Security Hub)](1.5-week5/)

**Week 6:** [Frontend learning and architecture design](1.6-week6/)

**Week 7:** [Midterm exam preparation](1.7-week7/)

**Week 8:** [Midterm exam and project setup](1.8-week8/)

**Week 9:** [Homepage creation and API integration](1.9-week9/)

**Week 10:** [AWS Cloud Mastery event and proposal writing](1.10-week10/)

**Week 11:** [Frontend restructuring and AWS Cloud Mastery #2](1.11-week11/)

**Week 12:** [Rekognition testing and final preparations](1.12-week12/)
