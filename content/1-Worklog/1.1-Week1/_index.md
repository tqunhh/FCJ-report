### Week 1 Achievements:

* Understand what AWS is and grasp basic service groups: 
  * Compute
  * Storage
  * Networking 
  * Database
  * ...

* Successfully created and configured AWS Free Tier account.

* Familiarized with AWS Management Console
  
* User and cost management 
  * Create and manage IAM User, Group and assign permissions for secure access
  * Set up Budget to monitor and control costs

* Install and configure AWS CLI on computer including:
  * Access Key
  * Secret Key
  * ...

* Understand and deploy networking on AWS
   * Grasp concepts and configure Amazon VPC
   * Know how to create Subnet, Internet Gateway, Route table, Security Groups

* Hands-on with EC2
   * Understand EC2 instance types and use cases
   * Practice creating EC2 instance, connecting via SSH/Session Manager
   * Create and configure NAT Gateway 
   * Manage EC2 security through Security Groups
