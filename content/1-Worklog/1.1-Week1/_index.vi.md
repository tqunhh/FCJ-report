### Kết quả đạt được tuần 1:

* Hiểu AWS là gì và nắm được các nhóm dịch vụ cơ bản: 
  * Compute
  * Storage
  * Networking 
  * Database
  * ...

* Đã tạo và cấu hình AWS Free Tier account thành công.

* Làm quen với AWS Management Console
  
* Quản lý người dùng và chi phí 
  * Tạo và quản lý IAM User, Group và phân quyền để truy cập an toàn
  * Thiết lập Budget để theo dõi và kiểm soát chi phí

* Cài đặt và cấu hình AWS CLI trên máy tính bao gồm:
  * Access Key
  * Secret Key
  * ...

* Hiểu và triển khai mạng trên AWS
   * Nắm khái niệm và cấu hình Amazon VPC
   * Biết cách tạo Subnet, Internet Gateway, Route table, Security Groups

* Thực hành với EC2
   * Hiểu được các loại EC2 instance và trường hợp sử dụng
   * Thực hành tạo EC2 instance, kết nối qua SSH/Session Manager
   * Tạo và cấu hình NAT Gateway 
   * Quản lý bảo mật EC2 thông qua Security Groups
